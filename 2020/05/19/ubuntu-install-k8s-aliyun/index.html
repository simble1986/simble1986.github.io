Node处于NotReady状态

node处于NotReady状态的原因有很多。可以一步一步处理

先查看错误原因：

k8s有些服务会在各个节点上启动，比如这里的proxy，flannel。

最常见的是ImagePull失败。比如master node上镜像拉取正常，而其他节点拉取失败。

解决方法

1. 在slave节点上手工拉取镜像
2. 将master节点上的镜像导入slave节点
  • 查看master节点上的镜像
  • 将镜像导出为文件
  • 将文件传输到slave节点上
  • slave节点上导入镜像
